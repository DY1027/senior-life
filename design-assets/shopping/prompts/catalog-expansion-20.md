# 생활상품 카탈로그 확장 이미지 20종

생성 방식: Codex 내장 이미지 생성 도구를 사용한 신규 생성.

공통 프롬프트 조건:

- 시니어 온라인 쇼핑 연습관에 쓰는 실물형 전자상거래 상품 사진
- 따뜻한 흰색 또는 아주 연한 회색의 이음매 없는 스튜디오 배경
- 제품 전체가 잘리지 않게 중앙 배치한 약간 위쪽의 3/4 시점
- 부드러운 자연 그림자와 현실적인 재질, 1:1 정사각형 구도
- 작은 상품 카드에서도 종류를 즉시 알아볼 수 있는 단순한 배치
- 브랜드, 로고, 인증마크, 상품명, 가격, 글자, 워터마크, 포장 상자 없음
- 사람이나 손을 넣지 않음

개별 생성 대상:

1. 흰색 블루투스 무선 이어폰 1쌍과 열린 충전 케이스
2. 회색 무선 마우스와 작은 USB 수신기
3. 흰색 충전식 동작감지 센서등과 자석 부착판
4. 회색 흡착식 욕실 미끄럼방지 매트
5. 흰색 욕실 안전손잡이와 고정 나사
6. 흰색 높이조절 샤워의자
7. 갈색 70cm 긴 손잡이 구두주걱
8. 검은 손잡이 대형 확대경
9. 베이지색 소파걸이 리모컨 정리함
10. 남색 여행용 정리 파우치 3종
11. 파란색 메모리폼 여행용 목베개
12. 청록색 접이식 보조가방과 접은 보관 주머니
13. 회색 투명 500mL 손잡이 물병
14. 남색 휴대폰 목걸이 스트랩과 투명 연결 탭
15. 노란색 성인용 경량 후드 우비
16. 흰색 충전식 휴대용 선풍기
17. 회색 충전식 손난로 2개와 충전 케이블
18. 남색·회색·베이지색 겨울 보온 양말 3켤레
19. 파란색 일주일 약 보관 파우치와 투명 보관통 7개
20. 회색 아코디언형 칸막이 서류 정리함

출력 관리:

- 생성 원본: `design-assets/shopping/originals/catalog-expansion/*.png`
- 사이트용 자산: `public/images/shopping/products/**/*.jpg`
- 사이트용 이미지는 `scripts/optimize-shopping-images.ps1`로 800×800 JPG 변환
