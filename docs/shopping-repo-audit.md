# 쇼핑 연습관 저장소 점검

- 구현 위치: 기존 Next.js 앱의 `/shopping` 경로. 별도 앱이나 로그인은 만들지 않는다.
- 기록: `localStorage`의 `dd-shopping-progress-v1`에 현재 단계와 완료 횟수만 저장한다.
- 결제: 연습 화면에는 결제·주소·전화번호 입력이 전혀 없다.
- 제휴: 연습 결과 다음의 제품 선택 기준 화면에서만 기존 쿠팡 파트너스 API를 사용한다. 공식 추적 링크가 없으면 광고 카드를 숨긴다.
- 이미지: `next/image`와 로컬 최적화 JPG를 사용해 모바일 레이아웃 이동과 데이터 사용량을 줄인다.
- 검색 노출: 허브와 미션 소개만 색인하며 연습 진행·결과 화면은 `noindex`와 robots 제외 대상으로 둔다.
